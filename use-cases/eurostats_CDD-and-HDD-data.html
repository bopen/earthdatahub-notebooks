Eurostat CDD & HDD data over Europe

The CDD and HDD data data set is created from the Eurostat datasets of cooling and heating degree days.

The Eurostat dataset was geoprocessed to rasterize and process the dataset.

The dataset can be easily integrated and visualized. For that the Python library xarray is used.

In[1]:
import xarray as xr

The dataset is defined.

In[2]:
dataset = "s3://eurostat/nrg_chddr2_a-19790101-20220101.zarr"

Then the data is read and the general dataset information is shown.

In[3]:
ds = xr.open_dataset(
    dataset,
    engine="zarr",
    chunks={}
)
